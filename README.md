## ▶️ Running the Programs

Clone the repository:

```bash
git clone https://github.com/Nipun-Lakshan/hackerrank-c-exercises.git
```

Navigate to the project directory:

```bash
cd hackerrank-c-exercises
```

Compile a C program:

```bash
gcc filename.c -o program
```

Run the executable:

```bash
./program
```
